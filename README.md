## Hướng dẫn sử dụng
1. Cài đặt **Mendix Studio Pro**
2. Clone hoặc tải repository này về máy
3. Import file `.mpk` vào Mendix Studio Pro
4. Chạy ứng dụng trong môi trường phát triển để trải nghiệm các chức năng đã triển khai

---

## Ghi chú
- Dự án được xây dựng cho mục đích học tập và nghiên cứu
- Hệ thống chưa triển khai đầy đủ các chức năng nâng cao như phân quyền chi tiết, bảo mật nâng cao hoặc tối ưu hiệu năng
